`/` 下的各个文件夹:

- `bin`: 常用二进制应用 (例如`ls, cd, cp...`等指令)
- `boot`: 系统引导加载程序
- `dev`: 设备文件
- `etc`: 系统配置文件
- `home`: 用户目录
- `lib`: 系统允许所需的库
- `media`: 挂载的可移动媒体设备(如, USB, CD, 等)
- `mnt`: 挂载其他存储设备或临时文件系统
- `opt`: optional. 存储“可选的“软件包和应用程序， 这里可选的表示非系统必须软件。也就是说如果装了其他软件一般装在这里
- `proc`: process, 提供了一种访问系统运行时各种各样信息的接口， 如进程信息(目录下的纯数字文件夹表示`pid`为这个数字的进程的信息)， 系统信息，内核参数等。**不会占用磁盘**。
- `root`: 系统管理员的主目录
- `run`: 与 `proc` 相似但完全不同，`run` 提供系统允许时的各种临时文件，这些文件一般会在重启时清空。存储在内存而非磁盘中。
- `sbin`: system binary, 系统管理和维护的二进制应用
- `srv`: service, 存储特定服务如 Web, FTP, git等数据
- `sys`: 提供内核对象和硬件设备的管理接口
- `tmp`: 临时文件，重启会清空
- `usr`: UNIX System Resources and UNIX Source Repository, 存放只读的用户数据和应用程序
- `var`: 存放运行时产生的可变数据，如logs， caches
